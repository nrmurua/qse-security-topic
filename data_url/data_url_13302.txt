Hi @anhdpham,
RX(-pi) is, up to a global phase, equal to X^-1, which is also equal to X for this particular gate. So, if I am not wrong, this is the expected behavior of unitary folding. Note that unitary folding maps a unitary G to G G^-1 G.
I agree that it would be nice to see X^-1 or  X instead of RX(-pi) in the printed output, but I don't think this depends on Mitiq. It looks more like PyQuil aspect.
Please reopen this issue if you disagree.
